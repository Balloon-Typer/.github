# 객체지향적 구조로 구현한 타이핑 게임 개발 프로젝트

## SINGLE MODE
##### level 1 : 제한시간 1분 30초, 우주배경
##### level 2 : 제한시간 1분 20초, 노을배경
##### level 3 : 제한시간 1분 10초, 지상배경
### 아이템 
##### 파란색 글씨 : BALLOON ITEM (풍선 +2, 풍선 -2) RANDOM
빨간색 글씨 : TIME ITEM (시간 +5초, -5초) RANDOM

##DUAL MODE
- 동일한 WIFI 공유기의 IPv4를 사용하여 두개의 노트북으로 사용 가능
###아이템
##### 파란색 글씨 : BALLOON ITEM (내 필드 풍선 +2, -2, 상대 필드 풍선 +2, -2) RANDOM
##### 초록색 글씨 : TRICK ITEM (REVERSE, BLIND) RANDOM
